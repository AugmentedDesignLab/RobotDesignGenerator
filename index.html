Understanding the Design Space of
(Web-based) Mixed Initiative Robot Design Tools


We will design a web-based mixed-initiative robot design tool. This tool will build upon existing WebGL technologies, Reactive planners, Answer Set Programming, numerical constraint solvers and generative grammars (L-systems). The output of the system will be a model description file, along with instructions that can be used to order the various parts, perform 3D printing of the chassis, and assemble the parts together to make a functional robot.
